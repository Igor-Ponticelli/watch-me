<h4>Como rodar a aplicação?</h4>
<ol>

  <li>Abrir a pasta no VsCode</li>
  <li>Abrir o terminal e rodar o servidor com "yarn server"</li>
  <li>Abrir outro terminal e rodar a aplicação com "yarn dev"</li>
</ol>
